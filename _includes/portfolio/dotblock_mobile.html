DotBlock Mobile

Javascript, Titanium Mobile, Ruby, Rails, MySQL, PHP

I created DotBlock Mobile for iPhone using Titanium Mobile. DotBlock Mobile allows clients to manage their DotBlock VPS and client information.

In order for DotBlock Mobile to edit and update client information, an API needed to be created. As a subproject, I created the DotBlock API for use by all DotBlock customers. I also created open source API clients in Bash, PHP, and Ruby.

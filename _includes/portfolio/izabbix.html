iZabbix

Ruby, Rails, Javascript, Titanium Mobile, MySQL

I created iZabbix, an internal iPhone app for ViaTalk for VoIP administrators to manage network alerts using Titanium Mobile and Ruby on Rails.

The iZabbix web application relays network alerts sent via XMPP to administrators' iPhones via push notification. My work with Apple Push Notifications has been extracted to a RubyGem, APND.
